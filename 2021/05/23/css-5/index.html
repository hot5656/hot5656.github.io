position: relative/absolute, top(bottom) left(right)使用百分比

  • 在相對定位中使用百分比時，參照的是最近一層父元素的高度與寬度
  • 在絕對定位中使用百分比時，參照的是最近一層父元素的高度與寬度，但父元素不能是static定位；若父元素是static定位，則會一直往上一級查詢，直至查詢到整個網頁的根元素html。
