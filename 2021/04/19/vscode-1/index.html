快速鍵

  • 重整程式排列 : shift-alt-f
  • 移動整行程式碼 : Alt + ↑ 或 Alt + ↓
  • 設註解 : ctrl+/
  • 跳至 Command line : ctrl+shift+p 或 F1
  • 跳至某一行 : Ctrl+G
  • 全螢幕切換 : F11
  • 多行編輯-選擇行 : alt+(mouse-left)
  • 多行編輯-選擇多行 : alt+(mouse-left) 移動位置
