快捷鍵

option + Up/Down : 行前後移
Ｃmd + X : 剪下
Cmd + C : 複製
Cmd + V : 貼上
Cmd + Z : 取消

Cmd + S : 存檔
Cmd + / : 註釋設定或取消
Ｃmd + Left/Right : 跳至行最前/最後
Cmd + F : 搜尋
Cmd + Ｕp/Down : 跳至檔案最前/最後
Cmd + ]/[ : 縮排加/減
control + G : 跳至指定行
Cmd + K : 清除終端機
control + Cmd + F : 全螢幕
shift + Cmd + P : 命令列
