### command 

+ nslookup : 查詢 IP address
``` bash
# local server IP
nslookup
# 查某一 host IP address
nslookup google.com
```

+ ls : 列出檔案目錄

<!--more-->
+ cd : 切換目錄

+ mkdir : 新增目錄

+ cp : copy 檔案,目錄

+ mv : 更改檔名或移動檔案位置

+ rm : 刪除檔案(-f 可刪除目錄)

+ rmdir : 刪除目錄

+ ping : 檢查某一 IP 的 device 是否存在

+ date : 列出日期時間

+ cat : 將檔案內容列出來

+ less : 檔案分頁顯示

+ touch : 更新檔案時間或產生檔案

+ echo : 顯示內容

+ man : command help
``` bash
man ls
```

+ grep : 從串流資料或檔案中，篩選出想要尋找的資料
