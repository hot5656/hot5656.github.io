### Connect to GitHub
#### Lovable 
``` bash
1. lovable connect to GitHub
2. Enter project --> GitHub connect --> Connect project --> add oginization --> select my account --> All repositories --> input password
```

#### Bolt AI 
``` bash
1. Enter Bolt AI project
2. select GitHub --> Authorize StackBlitz --> connect to GitHub --> enter repositories name
```

#### Replit 
``` bash
1. enter project --> search Git --> Settings --> GitHub  "sign in" --> Create Remote
2. search shell --> run "git push -u origin main"
```
