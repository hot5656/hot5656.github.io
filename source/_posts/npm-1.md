### Dump version

``` bash
npm -v
```

### Dump all npm installed models

``` bash
npm list
```

### 檢查 node_modules 有相關資安漏洞
``` bash
npm audit
```

### 自動修正相關漏洞
``` bash
npm audit fix
```

### 更新可更新的 node_modules
``` bash
npm update
```

### 清理 node_modules 中不需要的檔案
``` bash
npm prune
```
