### command
#### install
``` bash
pip install scrapyrt
```
##### 1st run

###### settings.py
+ set item count = 150
+ cache hold 3600s

``` py
# max item count
CLOSESPIDER_ITEMCOUNT = 150

# enable cache
HTTPCACHE_ENABLED = True
# every 3600s(1 hour) query once
HTTPCACHE_EXPIRATION_SECS = 3600
```
