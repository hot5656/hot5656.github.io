## python 100 days code save to GitHub

### git init + add .gitignore
``` bash
# git init 
cd python_100ds
git init
# create .gitignore
git init
```

### add file and commit 
``` bash
git status
git commit -m "first commit"
```

### push code to GitHub
GitHub repository 要先建立
``` bash
git remote add origin https://github.com/hot5656/python_100ds.git
git push -u origin master
```
