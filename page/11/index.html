CSS Box model

Box model 規範 HTML 每個元素(element)的基本設計規範
Box model 包含 content, padding, border and margin

  • content : 放文字或圖像的地方
  • border : 為 element 的邊框
  • padding : content 與 border 間的區域
  • margin : element 與鄰近 element 的距離
